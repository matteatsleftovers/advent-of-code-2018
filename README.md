# Advent Of Code 2018

Project for Kotlin tests and solutions for the [2018 puzzles](http://adventofcode.com/2018).

This project uses:

- Kotlin 1.3.10
- Gradle 5.0
- KotlinTest 3.1.10

# Instructions

## Running Solutions

All days:

    ./gradlew run

Specific day, e.g. 14:

    ./gradlew day14:run

## Running Tests

All days:

    ./gradlew test

Specific day, e.g. 14:

    ./gradlew day14:test
